Using Markdown in DEVONthink

DEVONthink에서 만들 수 있는 텍스트 파일은 Plain Text, Rich Text 파일입니다.

마크다운 파일을 쓰려면 외부에서 마크다운 파일을 만들어서 임포트해야 합니다.

이 과정을 애플스크립트로 자동화해서 쓰도록 합니다.

구현

/tmp 디렉토리에 blank.markdown이란 임시파일을 생성한 다음 DEVONthink에서 임포트하도록 합니다.

애플스크립트 편집기를 열어서 아래 코드를 입력합니다.

do shell script "touch /tmp/blank.markdown"

tell application id "com.devon-technologies.thinkpro2"
    import "/tmp/blank.markdown" name "Change this title"
end tell

do shell script "touch /tmp/blank.markdown"
 
tell application id "com.devon-technologies.thinkpro2"
    import "/tmp/blank.markdown" name "Change this title"
end tell

do shell script "touch /tmp/blank.markdown"

tell application id "com.devon-technologies.thinkpro2"
    import "/tmp/blank.markdown" name "Change this title"
end tell

메뉴 막대에서 사용하려면 ~/Library/Scripts/ 디렉토리에,

DEVONthink 메뉴에서 사용하려면 ~/Library/Application Support/DEVONthink Pro 2/Scripts/ 디렉토리에

스크립트 파일을 New Markdown이라는 이름으로 저장합니다.

저는 ~/Library/Scripts/DEVONthink Pro/, /Library/Application Support/DEVONthink Pro 2/Scripts/Data/ 밑에 파일을 두고 사용하고 있습니다.

단축키

시스템 설정 > 키보드 > 단축키 메뉴로 들어가서 DEVONthink 어플리케이션에 위에서 스크립트 파일명으로 지정한 New Markdown이란 제목으로 단축키(저는 Command-Control-M)를 지정합니다.

실행

  • 메뉴 막대 또는 DEVONthink 메뉴의 스크립트 메뉴에서 New Markdown을 선택해서 새 마크다운 파일을 만듭니다.
  • 위에서 지정한 단축키를 눌러 새 마크다운 파일을 만듭니다.

마크다운 훑어보기

Quick Look for Markdown in OS X

작성된 마크다운 파일을 쉽게 보기 위해서 Phil Toland씨가 만든 훑어보기 플러그인 QLMarkdown을 사용합니다.

설치

  1. 플러그인 파일을 내려 받습니다.
  2. QLMarkdown.qlgenerator 파일을 ~/Library/QuickLook 또는 /Library/QuickLook로 옮겨 놓습니다.

지원하는 파일 확장자

  • .md
  • .mdown
  • .markdown

실행

파인더에서 마크다운 파일을 선택하고 스페이스 키를 눌러 훑어보기를 합니다.
